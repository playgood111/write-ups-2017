## Smart devices

Some challenges required devices to borrow on site:

- Arduino boards
- FortiCam IP webcams
- Parrot Drone
- ReconJet smart glasses
- Nucleo boards with a washing machine simulator
- Nucleo boards with a fully functional weather station
